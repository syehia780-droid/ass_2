# 🎓 Student Grade Management System & 🎬 Cinema Ticket Booking System

A Java console-based application that includes two independent systems:

1. **Student Grade Management System**
2. **Cinema Ticket Booking System**

This project was developed to practice Java programming fundamentals, including arrays, methods, loops, conditional statements, input validation, and problem-solving.

---

## 📌 Features

### 🎓 Student Grade Management System
- Add student information.
- Store student grades.
- Calculate total and average grades.
- Assign grades (A, B, C, D, F).
- Display all student records.
- Search for a specific student.
- Validate user input.

---

### 🎬 Cinema Ticket Booking System
- Display available seats.
- Book cinema tickets.
- Cancel booked tickets.
- Prevent duplicate bookings.
- Show seat availability.
- Validate row and column input.
- Simple text-based user interface.
